# WhyOpen?

**WhyOpen** is a mindful Android application designed to help users break digital habits and reclaim their time. Instead of mindless scrolling, WhyOpen prompts you to pause and reflect on *why* you are opening a specific app.

## 🚀 Key Features

- **App Protection**: Choose which apps to "protect" from impulsive usage.
- **Mindfulness Prompts**: Before a protected app opens, WhyOpen presents a question or reflection prompt to help you stay intentional.
- **Focus Analytics**:
  - **Saved Time**: Track how much time you've reclaimed.
  - **Focus Streak**: Build a daily habit of mindfulness.
  - **Blocked Attempts**: See how many times you chose to stay away from distracting apps.
  - **Focus Score**: A unique metric to quantify your daily intentionality.
- **Privacy First**: All your usage data and app selections stay securely on your device. We do not collect or monitor your personal information.

## 🛠️ How it Works

WhyOpen utilizes several Android system features to provide a seamless experience:
- **Accessibility Service**: To detect when a protected app is about to be launched.
- **Overlay Permission**: To show the mindfulness prompts on top of protected apps.
- **Usage Access**: To provide accurate analytics and focus scoring.

## 🏗️ Getting Started

### Prerequisites
- Android Studio Ladybug or newer.
- Android device or emulator running Android 8.0 (Oreo) or higher.

### Building
1. Clone the repository:
   ```bash
   git clone https://github.com/draken0654-ui/whyopen.git
   ```
2. Open the project in Android Studio.
3. Build and run the `app` module on your device.
4. Enable the necessary permissions (Accessibility, Overlay, and Usage Access) from the app's dashboard.
